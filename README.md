## Installation (Development mode)

### 1. Download or clone the repository

### 2. Open the project's home folder

### 3. Start the JSON Server for the database

```bash
yarn db
```

or

```bash
npm run db
```

### 4. Start the application

```bash
yarn dev
```

or

```bash
npm run dev
```
